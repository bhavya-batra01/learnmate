# LearnMate 📚🤝

LearnMate is a modern student collaboration platform designed to help learners connect with compatible study buddies, plan study sessions, and achieve academic goals together. Built using HTML, CSS, and JavaScript, the platform provides an interactive and responsive interface focused on collaborative learning and productivity.



## ✨ Features

- 🤝 Smart study buddy matching system
- 📩 Connection request management
- 💬 Interactive chat interface
- 📅 Study session planner
- 🧠 Skill-based compatibility matching
- 👤 User profile customization
- 🖼️ Profile image upload support
- 📌 Saved connections dashboard
- 📱 Responsive and modern UI design
- ⚡ LocalStorage-based dynamic functionality



## 🛠️ Technologies Used

| Technology | Purpose                                   |
|------------|-------------------------------------------|
| HTML5       | Structure                                |
| CSS3        | Styling & Responsive Design              |
| JavaScript  | Dynamic Functionality & DOM Manipulation |
| LocalStorage| Data Persistence                         |
